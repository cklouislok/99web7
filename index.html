Project Title: Neighborhood Explorer

A user-friendly public web application that will help newcomers of a city to explore the the neighborhoods for finding suitable housing location.

Our Solution

Problem Statement

  • Ideal

    A user-friendly public web application that will help newcomers of a city to explore the neighborhoods for finding suitable housing locations.

  • Reality

    There are separate online maps with different thematic data from different sources such as crime maps, school maps, public amenity maps, etc., but there is no interactive map that is easy for the public to navigate and look for the different thematic information they want. Moreover, 3D map is rare in such online maps.

  • Consequences

    - People have to search for the information one by one on different platforms such as Google Maps, OpenStreetMap, and relevant data sources.
    - Without such an interactive map, people need to visit the neighbourhood physically to explore the surrounding environment, which is very inconvenient and time consuming.

  • Proposal

    Create a user-friendly hub with four thematic maps that are easy to navigate. First map displays public amenities, and allow users to search for points of interest within a certain radius of a location. Second map allows the users to define the location and the area of a circle to explore crime statistics of the intersecting neighborhoods in a bar chart format. Third application contains a 3D map for a closer to reality view. Fourth application provides summary statistics on the public amenities. (City of Toronto is chosen based on availability of data; Neighborhood statistics include public safety; Amenity data includes education, health services, transport, recreation, groceries)

  • Technology Used:

    ArcGIS Experience Builder, Dashboards, Server, ArcGIS Maps SDK for JavaScript (focus on Query Statistics by Geometry/Distance)
