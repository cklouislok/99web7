Interactive and Intuitive Visualization of Neighborhood Point, Line and Polygon Data

ArcGIS Experience Builder provides a user-friendly and interactive solution to strtegically organizing multiple thematic maps into a single, easily accessible hub. The constantly evolving and wide variaty of widgets available in Experience Builder empowers the integration and visualization of point, line, polygon, and 3D multipatch data in an interactive and harmonious manner, allowing users to explore an area through simple navigation of tradition 2D maps and closer-to-reality 3D scenes.

Featured function 1: Near Me widget - Locating nearby features and the closest feature of a location

The Near Me widget is the ideal solution to the shortcoming of ordinary online map application - if we want to search for schools in an area, we type in school in the search bar, and the application will display the search results within your current map extent. However, this is not good enough for someone who wish to explore the surround neighborhood when they are doing house-hunting.

By setting the analysis type as proximity, the Near Me widget finds and analyzes all public amenities within a specified distance of a defind location. In comparison, by setting the analysis type as closest feature, the Near Me widget finds and analyzes the closest public amenities within a specified distance of a defind location. This leads to the development of the two tools in the 2D Exploerer - Around Me and Closest to Me respecively.

Featured function 2: Synchronized 2D & 3D Visualization + 3D Analysis

In the 3D Explorer application, the 2D map and the 3D map operate in paralle - when the users zoom in, pan, or search for locations in one, the other map is triggered and zoom to the same map extent. This provides users with a greater flexibility in exploring both the planar position as well as the simulated stereoscopic environment of the locations. This could be a particularly useful function to explore features that is only available in an three dimensional environment, such as building height, building density, skyline, etc.

The daylight and shadow cast effects in the 3D Toolbox are valuable tools to analyze the sunshine conditions in a location at different time in a day, in addition to on different day within a year. The line of sight tool also enables the exploration of unobstructed views from a location. These collectively allow users to look for a location with a desirable amount and angle of sunlight, and views.

The fly controller provides an immersive experience to visualize the 3D surrounding environment around a point location or along a path.

Featured function 3: Embedding external resources

The Embed widget open the door for additional custom functinalities via a URL or code. The highly configurable structure of Experience Builder on the one hand facilitate a swift web development with customizable tools, but on the other hand limit the functionalities and possibilities of ordinary web development, such as via ArcGIS Maps SDK for JavaScript. With the Embed widget, developers can develope web pages and other appplications on other platforms, and embed the relevant URL or simple code into the web experience. In traditional web development environment, the embedment is achieved by iframe. In Experience Builder, it only costs several clicks.

In the final product, the content in the Crime Explorer and Statistics pages are developed using ArcGIS Maps SDK for JavaScript and ArcGIS Dashboard respectively, and they are embedded into the web experience.
